# 🎨 Fashion Agency Website (Responsive Clone)

A modern and fully responsive fashion agency website built using HTML, CSS, and SCSS.  
This project focuses on clean UI, responsive layouts, and professional design structure.

---

## 📌 Overview

This is a front-end responsive website inspired by modern fashion/creative agencies.

The project demonstrates:
- Responsive design (mobile → desktop)
- Clean UI/UX
- Section-based layout
- Real-world website structure

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- SCSS (Sass)  
- Remix Icons  
- Font Awesome  

---

## ✨ Features

- 📱 Fully Responsive Design (Mobile + Desktop)
- 🎨 Modern UI with dark theme
- 🧩 Section-based layout (7 views)
- 📊 Stats section (Clients, Projects, Followers)
- 👥 Team cards with hover effects
- 📸 Image-based content sections
- 🚀 Smooth layout transitions
- 🔥 Clean typography & spacing
